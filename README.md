# roi_tracker

## Burger Bros ROI Tracker

A Vite + React + TypeScript web application for tracking restaurant ROI, end-of-day operations, expenses, payroll, stock, and financial reports.

### Tech Stack
- React 18
- Vite
- TypeScript
- Tailwind CSS
- React Router
- Recharts
- Lucide React

### Setup Instructions
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`

### Development
Run the development server with:
```bash
npm run dev
```

### Build
Create a production build with:
```bash
npm run build
```

### Folder Structure
- `src/` - Application source code
  - `components/` - Reusable UI components
  - `pages/` - Route-level page components
  - `layouts/` - Application layouts (e.g., RootLayout)
  - `routes/` - Routing configuration
  - `styles/` - Global CSS
  - `utils/` - Constants and helper functions
- `docs/` - Project documentation
- `skills/` - Reusable agent workflows
